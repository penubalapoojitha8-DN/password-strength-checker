# 🔐 Password Strength Checker

## 📌 Description
A Python-based tool that evaluates the strength of a password using length, uppercase, lowercase, numbers, and special characters.

## 🎯 Features
- Checks password length
- Detects uppercase & lowercase letters
- Identifies numbers and special characters
- Classifies password as Weak / Medium / Strong

## 🛠️ Technologies Used
- Python
